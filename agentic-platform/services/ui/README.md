# UI Service

React TypeScript frontend for visualizing hyperparameter optimization in real-time.

## Features

- Real-time agent visualization
- Hyperparameter space exploration view
- Performance metrics charts
- WebSocket connection to optimiser service

## Development

```bash
# Install dependencies
npm install

# Run dev server
npm run dev

# Build for production
npm run build
```

## Tech Stack

- React 18
- TypeScript
- Recharts (visualization)
- Lucide React (icons)
- Tailwind CSS (styling)
